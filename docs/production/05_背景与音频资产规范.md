# 背景与音频资产规范

## 背景色调卡

- 数量：8 张。
- 基准尺寸：1920x1080。
- 格式：PNG 或 WebP；当前占位资产使用 PNG。
- 用途：提供地点氛围，不承担关键线索或可点击地图功能。
- 关键剧情信息不得只画在背景中而没有文本描述。

## 短音效

- 目标数量：10 到 15 个。
- 正式建议格式：WAV 或 OGG。
- 建议采样率：44.1 kHz 或 48 kHz，项目内保持统一。
- 音效用于 UI、选择确认、奖励、警示、战斗命中、存档等。
- 不需要环境声分层系统。

## 循环音乐

- 数量：3 首：村庄、野外、首领。
- 正式建议格式：OGG。
- 必须可无明显爆音地循环。
- 当前不做动态音乐切片和分层混音。

## 占位资产

`assets/` 中的占位文件只用于验证路径、尺寸、加载和 UI 布局，不代表最终美术或声音质量。替换正式资源时优先保持文件路径稳定。
