# 开发环境锁定

## 核心版本

- Godot：4.6.2 Standard，Windows x86-64。
- 脚本：GDScript，不使用 C#/.NET 版。
- Python：3.13.x，CI 基准使用 3.13。
- Python 开发依赖：以根目录 `requirements-dev.txt` 为准。
- JSON Schema：Draft 2020-12。
- 文件编码：UTF-8。
- Git 默认分支：`main`。

## 升级规则

新手村首个可玩版本完成前不主动升级 Godot 大小版本。只有以下情况允许升级：

1. 当前版本存在明确阻断开发的问题；
2. 修复在新版本中已经稳定发布；
3. 升级分支完成全量验证和存档兼容检查；
4. 制作方批准。

不得让 Codex 在任务过程中自行“顺手升级到最新版”。
