# 项目宪章

## 1. 产品定义

本项目制作Windows端、单机离线、以文字、选择、静态NPC立绘、简洁地点色调卡和文本回合战斗为核心的中文RPG。当前版本只制作“第七新手村”，目标完整游玩时长3—4.5小时。

《王者》是索达公司最新推出的沉浸式网络游戏。24岁的枫月以现实本名进入游戏，在登录过程中意外穿越到游戏世界。他此前没玩过、没参加内测，也不了解游戏剧情，只能走一步看一步。系统界面可以使用，但退出功能被锁定。

其他“冒险者”同样是来自现实、无法退出的玩家，在开发文档中归类为玩家类NPC；他们能复活、记得现实，但很少向陌生人谈现实。村民等原生角色归类为系统NPC。所有玩家和系统NPC都能看到系统，魔兽不能。

## 2. 当前篇章闭环

醒来并确认无法退出 → 接受并推进三份委托 → 调查兔群异常 → 与岚音合作 → 狼径冲突与藏宝线索 → 悬崖藏宝洞 → 天枢神泉首领 → 九坛泉水抉择 → 净化与离村。完整游戏最终保留“留在游戏世界”或“回到现实”的选择，新手村只埋伏笔，不提前解释穿越真相。

## 3. 核心体验

1. 完整可执行剧情：任务必须有逐节点正文、对白、选项、失败续接和回访，不以大纲代替剧本。
2. 选择有后果：改变至少一项关系、资源、战斗过程、地点状态、经济、奖励或后续文本。
3. 标准RPG战斗：需要技能、消耗品、观察和首领机制，理解机制后可稳定通过。
4. 人物沉浸：9名具名NPC使用独立半身立绘，有对白的无名角色使用3套通用立绘。
5. 低成本氛围：8张地点色调卡、10—15个短音效、3首循环音乐；无配音和动态美术。
6. 可恢复失败：主线失败转化为资源、时间、伤势或替代版本，尽量不强制读档。

## 4. 固定边界

- 平台仅Windows 10/11 64位，完全离线。
- Godot 4.x稳定版与GDScript；不依赖运行时大模型、数据库或在线服务。
- 主角固定为枫月，现实本名与游戏名相同，不提供改名。
- 枫月可善良、谨慎、功利、怀疑、冒险、冷淡或隐瞒，但不得主动残虐无辜、性胁迫、为娱乐折磨他人或走纯毁灭路线。
- 玩家类NPC可复活；系统NPC默认不复活。当前新手村具名系统NPC不得被AI新增永久死亡。
- 允许自由存读档和关键选择自动存档。
- 当前不制作主角大立绘、Live2D、口型、骨骼动画、动态换装、战斗角色动画、精细场景插画或配音。

## 5. 规模

8个主线、最多5个支线/关系任务、9名具名NPC、3套通用人物立绘、8个地点、等级上限10、永久状态建议不超过45个。总玩家可见文本目标38,000—50,000字；超出必须通过实际计时审批。
