# 核心玩法与数值规则

## 1. 核心循环

选择地点 → 阅读事件 → 对话、调查、采集、战斗或交易 → 写入状态 → 获得成长 → 解锁新地点或任务。没有自由行走和随机生成剧情；所有事件来自外置数据。

## 2. 属性

力量、体质、敏捷、洞察、幸运。初始值：力量8、体质9、敏捷8、洞察10、幸运20。幸运提高提示、暴击、特殊掉落和部分成功机会，但不能跳过主线首领。枫月不知道具体攻略，洞察只根据现场证据提供信息。

- 最大生命：`80 + 体质×12 + 等级×10`
- 基础攻击：`武器攻击 + 力量×2 + 等级`
- 基础防御：`护甲防御 + 体质 + floor(等级/2)`
- 暴击率：`min(30%, 5% + 幸运×0.5% + 装备加成)`
- 闪避率：`min(25%, 敏捷×0.5% + 幸运×0.2% + 装备加成)`
- 伤害：`max(1, 攻击×0.85—1.15 - 防御×0.6)`

## 3. 战斗

标准文本回合RPG。六种动作：普通攻击、防御、技能、物品、观察、撤退。不设置法力值，技能使用冷却和每战次数。行动顺序由敏捷和可保存随机种子决定。岚音自动行动，玩家每回合可设置进攻、防守或支援倾向。

普通战斗3—6回合，首领8—15回合；同时敌人不超过3名。每个首领只使用一个独特核心机制和2—3阶段。完整规则见`20A_战斗与检定完整规格.md`。

## 4. 失败

失败分为部分成功和失败续接，尽量不要求读档。枫月与玩家类NPC可复活并承担资源、经验债、时间或装备代价。系统NPC当前篇章不永久死亡。普通战败回到回响祭坛，主线进入重试、救援、受伤或次日版本。

## 5. 成长与经济

等级上限10，离村预期8—9级，全完成可达10级。装备槽、经验曲线、技能、背包、堆叠、商店和稀有奖励保底见`20B_成长经济与物品规则.md`。

## 6. 关系

岚音使用信任和好感双变量0—10。关系影响同行、额外对白、休息和告别，不影响主线通关。拒绝暧昧不扣主线奖励。

## 7. 时间片

`morning / afternoon / evening / night`四个离散时间片，只影响少数文本、恢复、价格和任务版本，不模拟完整日程，不让重要主线永久错过。
