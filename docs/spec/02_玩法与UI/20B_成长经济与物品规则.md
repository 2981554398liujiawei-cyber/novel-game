# 成长、经济与物品规则

## 1. 等级与经验

新手村等级上限10。预计主线玩家离村时达到8—9级，完成全部委托和战斗可达到10级。升级所需经验：

| 当前等级→下一等级 | 经验 | 累计经验 |
|---|---:|---:|
| 1→2 | 100 | 100 |
| 2→3 | 150 | 250 |
| 3→4 | 220 | 470 |
| 4→5 | 300 | 770 |
| 5→6 | 400 | 1170 |
| 6→7 | 520 | 1690 |
| 7→8 | 660 | 2350 |
| 8→9 | 820 | 3170 |
| 9→10 | 1000 | 4170 |

任务、探索和关键选择约提供总经验的70%，战斗约提供30%。禁止通过重复地点无限刷取经验；可重复战斗在首次后经验降为20%，直至相关任务阶段变化。

升级：最大生命按公式增长，生命恢复至新上限的50%以上，并获得2点可分配属性。属性重置当前版本不开放。

## 2. 技能与装配

技能最多装配4个主动技能；被动技能不占主动槽，但当前版本被动不超过4个。所有技能数据必须包含：ID、名称、类型、目标、冷却、每战次数、数值、状态、来源、升级规则和战斗文本。

新手村技能池：砍击、洞察、快刀手或坚守、岚音标记、梦游，以及锻造、炼药、采集三项非战斗技能。非战斗技能不得占战斗技能槽。

## 3. 装备槽

装备槽固定为：武器、副手、头部、身体、饰品一、饰品二。双手武器占用武器和副手。当前版本不做耐久、随机词缀、套装收集或外观换装。

品质：普通、精良、黄金、鬼器、神器。新手村的鬼器和神器只能作为封印、线索或不可完全发挥的物品，不能让数值越过当前平衡。

## 4. 背包与堆叠

初始普通背包30格；贪婪之戒提升至60格。任务物品单独存储，不占普通格。消耗品每格最多20，材料每格最多99，装备不堆叠。关键奖励在背包满时进入任务保管箱，保管箱在广场和交付界面可访问。

关键物品不得丢弃、出售或被死亡惩罚移除。普通物品丢弃前需要二次确认。

## 5. 商店与价格

基础买价来自物品表，卖价默认基础买价的50%。任务状态可施加价格修正，但总修正限制在-20%至+50%。药田生态使用-10%、0%、+20%三个正式档位。当前版本没有拍卖、玩家自由交易、动态供需或随机竞价。

## 6. 死亡代价

当前新手村采用简化原著规则：

- 普通怪物战败：损失当前金币10%，上限20金币；随机失去1件非关键消耗品；推进1个时间片；
- 首领战败：在普通代价基础上可增加装备临时损坏、NPC救援成本或首领阶段变化，但不随机删除关键装备；
- 玩家类NPC战败：通过对白和状态表现掉级、装备掉落、经验损失或恢复时间；
- 当前版本不直接降低枫月等级，避免短流程被迫刷怪；原著的掉级逻辑转译为经验债、资源损失和战斗状态代价。

## 7. 稀有奖励保底

嘟嘟兔面罩、贪婪之戒、金刚巨盾、梦游技能书和离村所需关键道具必须有确定获得路径。分支可改变物品来源、描述、附加效果或后续评价，但不能因道德选择永久失去核心系统功能。
