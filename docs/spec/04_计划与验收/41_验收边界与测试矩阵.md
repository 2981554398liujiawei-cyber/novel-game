# 验收边界与测试矩阵

## 1. 产品范围

Windows 10/11 64位、单机离线；第七新手村3—4.5小时；总玩家可见文本38,000—50,000字；8主线、最多5支线/关系任务、9具名NPC、3通用立绘、8地点色调卡、10—15音效、3音乐、等级上限10。

## 2. 世界与人物

- 枫月现实本名与游戏名一致，没玩过《王者》，不得表现攻略知识。
- 所有冒险者均为来自现实的玩家类NPC，知道现实、无法退出、可复活。
- 系统NPC可见系统但不知道现实，当前新手村具名系统NPC不得永久死亡。
- 魔兽看不到系统。
- 枫月可善良、功利、冷淡、怀疑、冒险或隐瞒，但无残虐无辜、性胁迫、娱乐性折磨或纯毁灭选项。

## 3. 数据静态验收

以下必须为0：Schema错误、重复ID、无效引用、未知状态、类型不匹配、范围越界、非法状态操作、任务依赖循环、不可达节点、无出口强制循环、无终止节点、未登记允许循环、`keep`无已显示立绘、表情不存在、运行manifest引用夹具、关键奖励缺失、占位语、双内容状态字段。

## 4. 功能矩阵

| 模块 | 必测 | 通过边界 |
|---|---|---|
| 新游戏 | 默认状态、姓名、退出锁定 | 不跳任务；枫月不表现攻略知识 |
| 三份委托 | 任意顺序、完成2项、完成3项 | 2项解锁兔王；第3项仍可完成；奖励不重复 |
| 战斗 | 敏捷顺序、防御、冷却、每战次数、观察、撤退、岚音倾向 | 种子可复现；机制说明清楚；无无限回合 |
| 失败 | 部分成功、玩家复活、NPC受伤、救援版 | 不永久卡主线；代价正确保存 |
| 关系 | 高低信任、拒绝暧昧 | 拒绝不惩罚主线，无越界选项 |
| 存档 | 手动、自动、关键选择、损坏恢复 | 可自由读档；同一检定种子一致；无静默丢档 |
| 立绘 | 具名、通用、缺图、keep/show | 正常内容不用缺图剪影；缺图不崩溃 |
| UI | 探索、对话、战斗、功能页，720p/1080p/DPI | 关键文本与按钮无遮挡 |
| 音画 | 8背景、音效、3音乐、静音设置 | 无资源阻断；音量分别可调 |
| Windows | 中文路径、只读目录、离线、三环境 | 解压可运行，存档写用户目录 |

## 5. 战斗难度

普通战斗需要合理使用技能或物品但不强制刷级；首领首次可能战败，观察机制并调整准备后可稳定获胜。只按普通攻击通关全部首领属于平衡缺陷；要求反复刷怪或依赖低概率暴击也属于严重缺陷。

## 6. 内容验收

任务文档可从开场演到结算；玩家选择有即时回应；失败有续接；回访引用状态；角色语言符合指纹；总字数和实测时长同时达标。只有`verified`任务为DONE。
