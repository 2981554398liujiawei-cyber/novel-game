# Windows运行环境与性能边界

## 1. 支持范围

- Windows 10 22H2与Windows 11，64位x86-64；
- 不支持32位Windows、Windows 7/8、移动端、macOS或Linux正式构建；
- 解压后运行，不要求管理员权限，不写入安装目录；
- 完全离线，启动和游玩不发起网络请求；
- 支持中文路径、含空格路径和非管理员账户。

## 2. 显示

目标1920×1080，最低1280×720；支持窗口、无边框全屏和系统DPI 100%—200%。宽高比低于16:9时优先保证正文、选项和战斗操作可见，允许折叠侧栏，不允许裁掉关键按钮。

## 3. 性能预算

- 冷启动至主菜单：普通SSD环境5秒内；
- 读档进入场景：2秒内；
- 普通节点切换：300毫秒内，不计逐字显示；
- UI目标60帧，无持续卡顿；
- 空闲内存建议500MB以内，峰值不超过800MB；
- 新手村完整压缩包建议350MB以内；
- 日志单文件上限10MB，最多保留5份。

## 4. 存档与设置位置

使用Godot用户目录，Windows对应`%APPDATA%/WangZheTextRPG/`。目录包含`saves/`、`backups/`、`settings.json`和`logs/`。手动档、自动档和关键选择档分开命名。原子写入：先写临时文件、校验后替换；每个槽保留最近一个备份。

## 5. 输入与无障碍

核心流程必须可用鼠标或键盘独立完成。字号18—32px、逐字显示可关闭、文本速度可调、重要提示不只靠颜色、音乐和音效音量分开控制。当前不承诺屏幕阅读器完整支持，但UI控件应使用清晰标签并避免图片承载唯一文本。

## 6. 构建验收环境

至少在三种环境验证：Windows 10 1080p/100% DPI、Windows 11 1440p/150% DPI、Windows 11 720p窗口/200% DPI。每种环境必须完成启动、新游戏、战斗、保存、退出、读取和通关关键切片。
