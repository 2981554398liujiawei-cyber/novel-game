# 世界观与终局契约

## 1. 现实入口

《王者》是索达公司最新推出的沉浸式网络游戏。枫月在首次进入时失去对现实设备的感知，醒来后已身处第七新手村。触觉、疲劳、伤痛和物品重量都是真实体验。他能打开系统，却无法退出或联系现实。

枫月没有游戏攻略知识。所有区域、人物、怪物和隐藏规则对他都是未知。任何正确判断必须来自可见证据、通用游戏经验、系统提示或他人说明。

## 2. 冒险者与原生居民

世界中的冒险者都是与枫月一样来自现实的玩家，记得现实、拥有完整系统界面、死亡后可复活，但所有人的退出功能都被锁定。开发文档称其为玩家类NPC，玩家可见文本只称冒险者或姓名。

原生居民是系统NPC。他们拥有连续记忆、情感、社会关系和死亡观，能看见任务、等级、公告等系统规则，并把玩家复活视为世界法则；他们不知道索达公司或现实。魔兽看不到系统。

## 3. 退出与终局

菜单显示`RETURN_CHANNEL_LOCKED`。新手村离村门只通往天梦大陆，不通往现实。完整游戏终局必须提供两项真实选择：继续留在游戏世界，或回到现实。两项都必须有长期关系、责任和代价基础，不能在结尾突然出现。

## 4. 死亡

玩家和玩家类NPC会经历倒地、尸体保留或延迟复活，并承担经验、资源、装备或时间代价。当前版本为短流程，枫月不直接掉级，改用经验债、金币、消耗品、时间片和任务版本体现原著死亡惩罚。

系统NPC默认不能自动复活。当前新手村没有原著指定的具名系统NPC死亡，因此只允许受伤、昏迷、被困或暂离。后续永久死亡必须先完成原著命运映射和功能替代设计。

## 5. 玩家不常谈现实

现实话题没有被系统禁止。玩家很少谈论，是因为退出问题暂时无解、身份隐私、陌生人不信任和生存压力。关系建立后可以谈现实，但不能把每次初见写成现实背景说明会。

## 6. 当前谜团

当前版本只提出而不解答：索达公司是否知道异常、所有玩家为何同时无法退出、现实时间是否流动、系统NPC为何拥有完整人格、神泉和复活点为何能锚定玩家意识。不得在新手村提前给出最终答案。
