# 任务依赖图与章节状态矩阵

## 1. 说明

本文件把当前第七新手村剧情母稿转换为可编程依赖，不重新设计剧情。`NV_MAIN_002《三份委托》`是一个主线阶段，内部包含三个可并行子目标，不再使用旧版“黄金铁锤/黄金药鼎/二十条虎尾”作为当前委托定义。

## 2. 主线依赖

```text
NV_MAIN_001
  └─ NV_MAIN_002
       ├─ commission_hanshi_trial
       ├─ commission_suzhi_herb_basket
       └─ commission_guchangchuan_boundary_stones
            任意完成2项
               └─ NV_MAIN_003
                    └─ NV_MAIN_004
                         └─ NV_MAIN_005
                              └─ NV_MAIN_006
                                   └─ NV_MAIN_007
                                        └─ NV_MAIN_008
```

可选剧情和岚音关系内容在各自触发窗口并行，但必须以完整剧情母稿和后续`complete_script`为准，不在本依赖图中凭旧ID强行冻结。

## 3. NV_MAIN_002状态

| 状态 | 条件 | 作用 |
|---|---|---|
| `not_started` | 未接取 | 三项子目标不可追踪 |
| `active` | 已接取，完成数0—1 | 可自由执行三个子目标 |
| `qualified` | 完成数达到2 | 解锁`NV_MAIN_003`，第三项仍可继续 |
| `completed` | 三项全部完成 | 启用完整回访和额外结算 |

状态变量：

- `quest.nv_main_002.status`
- `quest.nv_main_002.completed_commissions`
- `quest.nv_main_002.commission.hanshi_trial.status`
- `quest.nv_main_002.commission.suzhi_herb_basket.status`
- `quest.nv_main_002.commission.guchangchuan_boundary_stones.status`

`NV_MAIN_003`程序前置必须检查`completed_commissions >= 2`，不能只检查`NV_MAIN_001`。第三项未完成不阻塞兔王事件，也不阻塞离村；离村时必须根据完成状态播放不同回访或结算。

## 4. 子目标责任

| 子目标 | 教学/剧情责任 | 必须写入的关键结果 |
|---|---|---|
| 韩石的试刃 | 攻击、防御、观察、初始武器选择 | 初始武器、基础战斗教学完成、黑铁异常线索（若触发） |
| 苏芷的药篮 | 采集、药品、谨慎调查 | 采集质量、药品奖励、蓝色异常粉末（若发现） |
| 顾长川的界石 | 地点调查、系统异常线索 | 界石异常、顾长川态度、神泉伏笔 |

完成数只在子目标首次从未完成变为完成时增加，重复交互不得重复计数。

## 5. 强顺序、并行和临时关闭

| 内容 | 顺序规则 | 关闭规则 | 防卡死规则 |
|---|---|---|---|
| 主线001 | 强顺序第一项 | 不关闭 | 新游戏直接进入 |
| 主线002三个子目标 | 可并行 | 战败/资源不足可暂缓 | 资源刷新、替代交付或休息后继续 |
| 主线003—008 | 强因果顺序 | 可因战败、受伤或时间片临时关闭 | 必须提供等待、救援、重试或替代版本 |
| 可选剧情 | 以母稿触发窗口为准 | 可错过或转回访版 | 不得影响主线通关 |
| 关系内容 | 信任阈值和主线阶段 | 阈值不足不开放 | 不影响主线和离村 |

## 6. 失败版本

任务失败不得简单写成永久`failed`并封死后续。需要暂停时使用可续接状态并记录`continuation_id`，指向救援、休息、补给、次日或受伤版入口。

玩家类NPC倒下后可按世界规则复活并承担代价；系统NPC的永久命运遵循原著/制作方正典。当前新手村无确认永久死亡节点时，只允许受伤、暂离或功能替代。

## 7. 回访矩阵最低要求

每个主线完成后至少更新：任务相关NPC对白、一个相关地点文本、一个世界状态或价格/公告反馈。关键分支还必须在至少一个后续主线中被引用，不能只改变当前任务最后一句话。
