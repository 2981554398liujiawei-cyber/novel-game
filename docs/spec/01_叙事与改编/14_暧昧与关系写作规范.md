# 暧昧与关系写作规范

## 1. 内容尺度

当前版本允许：互相打趣、关心伤势、短暂靠近、请求触碰、牵手、拥抱、含蓄表达在意、轻度吃醋和章节结尾的关系承诺。不写露骨性描写、强迫亲密、趁昏迷或受伤偷吻、以奖励换亲密、拒绝后纠缠、把角色身体当作物品评价。

## 2. 年龄与同意

岚音明确22岁。所有可发展关系角色必须在人物数据中记录`age >= 20`与`romance_eligible: true`。任何身体接触选项必须由玩家主动选择，并在文本中获得明确或自然可判断的同意。若岚音拒绝，立即尊重，不扣主线奖励，不重复施压。

## 3. 双变量模型

- `relation.lanyin.trust`：0—10，表示对枫月判断与承诺的信任。
- `relation.lanyin.affection`：0—10，表示私人好感。

可靠行动主要增加信任，温和而尊重的表达增加好感。暧昧场景最低要求：信任≥5且好感≥3。拥抱选项要求信任≥6、好感≥4；若条件不足，替换为“并肩站一会儿”而不是显示灰色羞辱玩家。

## 4. 对话示例方向

合格：岚音先询问是否可以替枫月包扎；枫月可答应、自己处理或用玩笑缓和。岚音在枫月冒险后说“下次先告诉我你的计划，我不喜欢替人收尸”，枫月可以认真道歉或轻松回应。

不合格：岚音因为枫月击败首领自动爱上他；枫月未经同意拥抱或亲吻；拒绝暧昧导致关键装备消失；用“征服”“占有”描述关系完成。

## 5. 当前篇章关键节点

1. 药田初见：互相质疑，只有微弱吸引，不立刻亲密。
2. 面罩玩笑：岚音可说“至少它比你刚才的计划可爱”，建立轻松语气。
3. 狼径包扎：她先问“我碰你的手臂可以吗？”玩家有三种尊重选择。
4. 火光问答：谈“家”和过去的错误，是关系核心，不写身体描写。
5. 神泉战后：岚音主动扶住枫月；是否进一步握手或拥抱由玩家选择。
6. 离村告别：岚音不承诺永远跟随，只承诺在天龙城的北门等一个答复，为未来留下空间。
