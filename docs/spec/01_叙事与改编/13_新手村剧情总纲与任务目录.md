# 新手村剧情总纲与任务目录

## 1. 总体节奏

第一幕“陌生规则”约35分钟：枫月醒来、确认无法退出、学习界面并接受三份委托。第二幕“幸运不是答案”约70分钟：调查嘟嘟兔事件，与岚音建立合作，理解选择会改变生态与人。第三幕“规则中的人心”约60分钟：救援天火冷魂、面对山狼王与风云三人组、获得藏宝图。第四幕“神泉试炼”约70—100分钟：藏宝洞、黄金战熊、九坛泉水的诱惑、净化仪式与离村结算。

## 2. 主线任务

### NV_MAIN_001《没有退出键》

枫月在回响祭坛醒来，系统以他的现实本名“枫月”完成登记，职业为见习战士。他此前没玩过《王者》，只能通过系统和现场逐步确认规则。教程引导玩家查看状态、历史记录和地点。玩家尝试退出时看到`RETURN_CHANNEL_LOCKED`。顾长川出现，询问枫月是否记得来路。玩家可坦白“我不属于这里”、只说失忆或反问村长；决定`relation.chief.trust`初值，但都进入后续。任务结束时获得木刀、粗布护甲和洞察术。

### NV_MAIN_002《三份委托》

顾长川要求枫月先用三份实际委托熟悉第七新手村。玩家可按任意顺序推进三个子目标：**韩石的试刃**负责基础攻击、防御、观察与初始武器选择；**苏芷的药篮**负责采集、药品和蓝色异常粉末线索；**顾长川的界石**负责地点调查、结界异常与神泉伏笔。至少完成两个子目标后，`NV_MAIN_002`进入`qualified`并触发北药田危机；三项全部完成后进入`completed`并获得完整回访与额外结算。第三项可在兔王事件之后补完，不得阻塞后续主线。三项使用主线子目标状态追踪，不强制拆成三个独立正式支线。

### NV_MAIN_003《兔王不该出现》

枫月在北药田遇到岚音。巨型兔群被捕兽夹和药物驱赶，嘟嘟兔并非主动袭村。玩家可安抚并调查、直接狩猎、或收集证据揭露捕兽者。三条路线均有战斗或检定，但结果不同：生态稳定可降低药价并提升岚音信任；狩猎获得更多即时材料但药价上涨；揭露捕兽者能得到最佳村庄状态和后续证据。嘟嘟兔面罩必须在所有路线可获得，但来源和描述不同，避免道德路线损失关键玩法装备。

### NV_MAIN_004《狼影中的同伴》

枫月和岚音前往西北狼径，先遇到被魔化刺猬王压制的法师天火冷魂。玩家可帮助、观察弱点后帮助或绕开；绕开不会卡主线，但失去友方关系。随后山狼王出现，风云三人组企图抢夺战果。玩家可利用山狼王制造混乱、公开警告、或正面迎战。战斗后获得升级宝石、狼锋和藏宝图线索。岚音会根据玩家是否保护倒地者评价枫月。

### NV_MAIN_005《悬崖下的宝藏》

藏宝图指向一次性洞窟。黑鼠王擅长黑暗与致盲。玩家可使用洞察术识别发光苔藓、消耗火把降低致盲，或硬战。奖励为贪婪之戒，扩大背包并提高金币获取；若背包已满，奖励直接进入任务保管箱。洞窟消失前出现一行异常系统文字：`外部坐标不可用`，强化穿越伏笔。

### NV_MAIN_006《天枢神泉》

王五在枫树林等待，说明神泉领域和三年前的失败。岚音可在信任达到3时同行。黄金战熊恰拉拉分为三阶段：沉睡突袭、熊扑与大地震击、低生命进化为黄金形态。玩家可使用绕树、陷阱标记或正面防御三种战术；战败触发王五救援版，损失资源并获得“受伤的恰拉拉”重试版本。胜利得到金刚巨盾（封印未鉴定）、战熊盔甲、梦游技能书和千年熊胆。

### NV_MAIN_007《九坛泉水》

玩家装取九坛未净化泉水。可选择直接喝、私藏一坛、完整交回或先让岚音检查。直接饮用会触发严重负面状态和救治，不造成角色删除；私藏会被后续净化数量检查发现并影响村长信任；完整交回提升村庄与人物关系。王五传授采集术并交付鹰之勋章，留下未来山地之王线索。

### NV_MAIN_008《离村之前》

枫月分别把泉水送给韩石和苏芷，取得离火之珠和寒冰之珠，由顾长川完成净化。玩家获得一坛净化泉水并提升基础属性。岚音在祭坛边询问枫月是否真的有“另一个家”。玩家可坦白、含糊或转移话题；信任与好感达到阈值时，可选“请求拥抱”“握住她的手”或“只说再见”，三项均为自愿且不影响主线。离村仪式显示：`RETURN CHANNEL: LOCKED / WORLD EXIT: TIANLONG GATE`。章节在枫月决定前往更大世界寻找答案时结束。

### 可选剧情与关系内容

制作方提供的新手村剧情母稿包含5个可选剧情，但当前生产启动包不擅自替它们冻结正式ID。它们必须在逐任务拆分时按照母稿实际内容登记，并满足`17_完整任务剧本与对白数据规范.md`。任何旧文档里的`NV_SIDE_001—003`黄金铁锤/黄金药鼎/二十条虎尾，不再作为当前三份委托的正式定义。

岚音关系内容仍为可选，不影响主线通关；具体任务ID在完整剧本冻结时确定。

## 4. 分支预算

普通任务最多3个重要选择，永久改变世界的选择最多1处；多数路线在1—3个节点后汇流，但保留关系、奖励、价格或对白差异。当前篇章允许三种区域结算标签：`守护者`、`逐利者`、`揭密者`。标签影响章节结语，不阻止离村。

## 5. 必须回收的伏笔

- 退出键灰色：在任务1出现，任务5强化，任务8确认仍锁定。
- 顾长川看见异常系统辉光：任务1暗示，任务8用一句“你的系统辉光和其他冒险者不一样”强化。
- 岚音关于家的问题：任务3初次试探，关系任务深入，任务8形成情感回响。
- 天枢神泉与世界稳定：任务3魔物异常、任务6揭示、任务8恢复。

## 6. 任务顺序、并行与版本规则

- `NV_MAIN_001`必须最先完成。
- `NV_MAIN_002`中的三个子目标可任意顺序进行；完成两项后`quest.nv_main_002.completed_commissions >= 2`，药田异常可出现；第三项仍可在兔群事件前后完成并使用不同回访文本。
- `NV_MAIN_003—008`保持因果强顺序；支线和关系任务可以在开放时段并行。
- 任何任务因战败、NPC受伤或物品不足而关闭时，必须提供等待、替代交付或救援版，主线不可永久卡死。
- 每个主要地点至少准备首次、进行中、战败后和完成后四类文本版本。
