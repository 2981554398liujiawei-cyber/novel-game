# 第七新手村区域圣经

## 1. 区域功能

第七新手村承担教程、世界规则介绍、第一段关系建立、第一次重要道德选择、稀有装备爽点、首领战和穿越伏笔。区域不是自由地图，而是由地点节点和旅行选择组成。旅行会推进时间片并触发已编写事件，不随机生成剧情。

## 3. 区域视觉基调

新手村整体采用朴素东方幻想边境聚落风格：木石建筑、旧布旗、铁器和草药器具，色彩以土褐、青灰、药草绿为主；天枢神泉加入金色与冷蓝魔法光。NPC立绘服装应来自同一地区物质条件，即使是外来冒险者也不能显得像另一个画风。首版固定制作8张简洁地点色调卡，对应本表8个地点，可采用模糊背景、纹理或低细节场景，不制作精细场景插画。配置3首循环音乐：村庄、野外、首领；10—15个短音效；不配音、不做分层环境声或动态音乐。

## 4. 区域状态

- `world.nv7.protection`: `weak` / `recovering` / `stable`。
- `world.nv7.rabbit_ecology`: `unknown` / `stable` / `damaged`。
- `world.nv7.herb_price_modifier`: -20至50的整数百分比。
- `world.nv7.spring_restored`: 布尔值。
- `world.nv7.poacher_exposed`: 布尔值。
- `world.nv7.rival_status`: `unknown` / `warned` / `defeated` / `reconciled`。

## 5. 环境变化

开场时广场热闹但商店缺货，北方道路有灰雾。兔王事件后，药田根据结果恢复或涨价；山狼王事件后，广场公告出现枫月或“无名冒险者”的传闻；神泉恢复后，灰雾消散、商店补货、战败返回惩罚降低，并开启离村仪式。

## 6. 文本与立绘配合

文本应在系统提示、动作描写、对话和玩家内心之间交替。单段建议不超过120个汉字；连续说明不超过3段，随后必须提供交互或选择。NPC首次登场可用一段简短外观描写，后续不反复复述立绘已经表达的信息。系统提示使用统一方括号格式，例如：`【洞察：草叶上的血迹不是兔类留下的。】`。不要用镜头、动画或地图坐标描述替代文本游戏可执行信息。
