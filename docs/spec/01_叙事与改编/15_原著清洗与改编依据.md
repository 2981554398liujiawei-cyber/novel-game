# 原著清洗与改编依据

本项目不向开发工具提供小说正文。本文件只记录已经提炼出的新手村剧情资产和清洗结论，开发工具应把本包中的改编设定视为唯一可执行正典。

## 1. 清洗结论

源文本已识别为GB系编码，转码时仅出现1个不可恢复字符；采集网站声明、网址、字数、求票和重复推广行不属于剧情。章节标题存在缺号、并入上一标题和重复编号，已在`90_审阅附件/原著章节标题清洗索引.csv`中整理为1—1273连续索引。没有独立标题的章节使用概括标题或“承接章节”标题，标题仅用于追溯，不作为游戏文本。

## 3. 改编原则

保留剧情功能、稀有发现、成长节点、装备、技能、怪物和区域因果，不复制原文句段。删除现实生活场景，把“登录游戏”改为“穿越后醒来”；把单纯刷怪改为有限的调查、选择和文本战斗；把不合适的亲密或性化表达改为成年角色之间自愿、克制的暧昧。

## 4. 当前正典命名

主角统一为“枫月”；区域统一为“第七新手村”；游戏世界统一为《王者》与天梦大陆。原始章节中的其他主角名、编号新手村和现实人物不进入玩家可见文本，也不进入程序数据。
