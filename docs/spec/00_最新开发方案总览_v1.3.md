# 《王者》新手村文本RPG 最新开发方案总览 v1.3

## 产品

Windows 10/11 64位单机离线文本RPG，Godot 4.x。当前只制作第七新手村，3—4.5小时，总玩家可见文本38,000—50,000字。采用探索、对话、战斗三种UI模式，9名具名NPC独立立绘、3套通用立绘、8张地点色调卡、10—15个短音效和3首循环音乐。

## 世界规则

《王者》是索达公司最新推出的沉浸式游戏。枫月现实本名就是枫月，此前没玩过游戏，在首次进入时穿越并发现退出被锁。所有冒险者都是来自现实、无法退出、可复活的玩家类NPC；他们知道现实但很少向陌生人谈。原生居民是系统NPC，能看见系统但不知道现实。魔兽看不到系统。

玩家类NPC按原著死亡规则简化为延迟复活和资源/经验/装备代价。系统NPC默认不能复活；当前新手村没有原著安排的具名系统NPC死亡，因此不得新增永久死亡。

## 主角边界

枫月可善良、谨慎、功利、怀疑、冒险、冷淡或隐瞒，也可私藏和拒绝救援；但不能主动残虐无辜、性胁迫、为娱乐折磨NPC或走纯毁灭路线。他没有攻略知识，所有判断必须从证据和试错获得。

## 战斗

标准RPG难度，无魔法值；技能使用冷却和每战次数。按敏捷和可保存随机种子行动；防御持续至下一行动并减伤40%；岚音自动行动，玩家每回合可给进攻/防守/支援倾向；普通调查条件满足即成功，重要检定使用可保存种子；失败转为部分成功或续接；每个首领一个核心机制和2—3阶段。

## 已修复的数据问题

- 正式默认状态不再预设任务完成，测试前置移入`fixtures/`；
- `NV_MAIN_003`检查三份委托完成数≥2；
- 苏芷首次对白改为`portrait_action=show`；
- 内容状态统一为`content_status`；
- 任务16类字段结构化；
- 补齐状态、地点、物品、技能、敌人、战斗、任务依赖、存档和manifest Schema；
- 校验器检查类型/范围、任务依赖、终止路径、允许循环、立绘状态流和夹具隔离。
