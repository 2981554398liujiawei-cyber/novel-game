# 全剧情任务树 v0.2 接入状态

原始来源：`01_全剧情任务树_v0.2.docx`。

状态：`draft_outline / future_roadmap_only`。

该任务树覆盖新手村到终局的长期路线，保留未来篇章方向、人物、地点、任务钩子和主题参考价值，但它产生于更早的开放世界设计阶段，其中包含当前项目已经取消的机制和世界规则。

## 当前禁止直接继承的内容

- 7×5战棋；
- 自由移动开放世界；
- 大规模后台模拟玩家；
- 把其他冒险者描述为“模拟玩家”；
- 与当前第七新手村剧情母稿冲突的ACT00任务结构；
- 未经当前规格重新审查的实时经济、帮派、拍卖和大型世界模拟。

## 可保留的价值

- 后续区域的大方向和篇章顺序；
- 原著重要角色、地点、神器和事件的覆盖提醒；
- 未来区域的主题冲突和对白钩子；
- 防止后续改编遗漏关键原著资产。

任何未来区域启动时，必须重新建立该区域的正式范围单、区域圣经、完整剧情母稿和数据预算，不能直接把本任务树交给Codex实现。
