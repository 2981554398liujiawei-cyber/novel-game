# src/AGENTS.md

本目录只存运行时代码。

- 严格保持模块职责边界，不把业务状态塞进 UI。
- 正式对白、任务条件、奖励、NPC 关系结果不得硬编码在 GDScript。
- 禁止联网相关类、远程服务和运行时 AI 调用。
- 新模块优先通过明确接口和信号协作，不创建万能单例。
- 任何随机逻辑必须可注入或保存种子，保证测试可复现。
- 内容加载错误必须显式失败并给出可定位信息，不允许静默忽略。
- 修改公共行为时同步更新测试和对应规格。
- 任务完成前至少执行 `scripts/test.ps1`；涉及启动流程时再执行 `scripts/smoke_test.ps1`。
